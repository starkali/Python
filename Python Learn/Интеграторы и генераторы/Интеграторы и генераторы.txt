В Python, интеграторы (iterators) и генераторы (generators) - это специальные объекты, которые позволяют перебирать наборы данных поэлементно.
Интеграторы являются объектами, которые реализуют протокол итератора,
который включает в себя метод __next__ для получения следующего элемента и метод __iter__ для создания интегратора.


Генераторы, с другой стороны, являются функциями,
которые используют ключевое слово yield для генерации последовательности значений.
Вызов функции-генератора создает объект-генератор, который может быть использован как интегратор.


Использование интеграторов и генераторов позволяет эффективно итерировать большие наборы данных,
избегая хранения всех данных в памяти
