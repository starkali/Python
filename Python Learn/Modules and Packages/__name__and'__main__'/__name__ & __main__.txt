`__name__` - это встроенная переменная в Python, которая указывает на имя текущего модуля.
Когда вы запускаете скрипт Python как основной процесс, `__name__` принимает значение `__main__`.

`__main__` - это специальное имя для основного модуля в Python, т.е. модуля,
который запущен ваш скрипт. Это означает, что вы можете использовать `if __name__ == "__main__"`: для задания кода,
который должен выполняться только при запуске этого файла, а не при импорте его как модуля.

Пример:

def function_1():
    print("This is function_1")

def function_2():
    print("This is function_2")

if __name__ == "__main__":
    function_1()
    function_2()
