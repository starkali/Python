Modules and Packages в Python это единицы организации кода.
Модуль - это файл с кодом Python, который может содержать функции, переменные и классы.
Пакет - это коллекция модулей, которые группируются вместе для улучшения организации кода и повышения эффективности разработки.

1) Создайте модуль: создайте файл с расширением .py и напишите в нем код.
2) Импортируйте модуль: в другом файле или в интерпретаторе, напишите import и имя модуля, чтобы использовать его код.
3) Создайте пакет: создайте директорию, содержащую модули и/или другие пакеты. Создайте файл __init__.py внутри директории, чтобы обозначить ее как пакет.
4) Импортируйте пакет: в другом файле или в интерпретаторе, напишите from и путь к пакету, чтобы использовать его код.
