Shelve - это модуль в Python, который позволяет сохранять объекты Python в файле как словарь и получать доступ к ним в будущем. Шаги работы с модулем shelve:

1) Импортировать модуль shelve.
2) Открыть файл с помощью функции shelve.open().
3) Сохранить объекты в файл с помощью присваивания.
4) Закрыть файл с помощью метода close().

Пример использования:

import shelve

# Открыть файл
s = shelve.open('test_shelf.db')

# Сохранить объекты
s['key1'] = [1, 2, 3]
s['key2'] = {'a': 1, 'b': 2}

# Закрыть файл
s.close()
