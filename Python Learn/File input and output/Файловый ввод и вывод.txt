1) Открытие файла: Чтобы прочитать или записать файл, вы должны сначала его открыть с помощью функции open().
Вы можете указать режим, в котором вы хотите открыть файл: "r" для чтения, "w" для записи, "a" для добавления в конец файла.

f = open("file.txt", "r")


2) Чтение файла: Чтобы прочитать файл, вы можете использовать методы, такие как `read()`, `readline()` или `readlines()`.

contents = f.read()


3) Запись в файл: Чтобы записать в файл, вы можете использовать метод `write()`.

f = open("file.txt", "w")
f.write("Hello, World!")


4) Закрытие файла: После завершения работы с файлом важно закрыть его с помощью метода `close()`.

f.close()


5) Контекстный менеджер: Вместо ручного закрытия файла вы также можете использовать контекстный менеджер,
чтобы автоматически закрыть файл после завершения работы с ним.

with open("file.txt", "r") as f:
    contents = f.read()

В данном примере, файл будет автоматически закрыт после выхода из блока `with`.
