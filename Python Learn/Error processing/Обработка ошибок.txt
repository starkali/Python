В Python, обработка ошибок осуществляется с помощью конструкции try/except:

1) Оберните код, который может вызвать ошибку, в блок try.
2) Затем, в блоке except, укажите тип ошибки, который вы хотите обработать.
3) Код внутри except выполняется, если происходит ошибка, указанная в except.

Например:

try:
    # Код, который может вызвать ошибку
    result = 10 / 0
except ZeroDivisionError:
    # Код, выполняемый в случае ошибки ZeroDivisionError
    print("Деление на ноль!")
