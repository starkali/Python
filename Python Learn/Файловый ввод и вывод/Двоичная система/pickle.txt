Pickle - это модуль Python, который позволяет сериализовать и десериализовать объекты Python.
Это означает, что вы можете сохранить состояние объекта в файл и позже восстановить его.

Шаг 1: импортируйте модуль pickle:

import pickle


Шаг 2: Сериализация объекта:

# Создаем словарь, который хотим сериализовать
data = {1: "Hello", 2: "World"}

# Открываем файл в режиме "wb" для записи байтов
with open("data.pkl", "wb") as f:
    # Используем функцию dump для сериализации объекта в файл
    pickle.dump(data, f)



Шаг 3: Десериализация объекта:

# Открываем файл в режиме "rb" для чтения байтов
with open("data.pkl", "rb") as f:
    # Используем функцию load для десериализации объекта из файла
    data = pickle.load(f)
